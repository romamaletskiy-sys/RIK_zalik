# Фінансист 💰

Мобільний додаток для управління особистими фінансами, розроблений на React Native (Expo).

## Функціональність

- **Баланс** — відображення загального балансу, сумарних доходів і витрат
- **Транзакції** — додавання витрат і доходів з вибором категорії та описом
- **Категорії витрат** — Їжа, Транспорт, Розваги, Покупки, Здоров'я, Освіта, Житло, Інше
- **Категорії доходів** — Зарплата, Фріланс, Подарунок, Інші доходи
- **Статистика** — кругова діаграма витрат за категоріями з детальним розбивком
- **Збереження даних** — всі транзакції зберігаються локально на пристрої (AsyncStorage)

## Технічний стек

| Технологія | Призначення |
|---|---|
| React Native + Expo | Мобільний фреймворк |
| React Navigation | Навігація між екранами |
| React Context API | Управління станом |
| AsyncStorage | Локальне зберігання даних |
| react-native-chart-kit | Кругова діаграма |
| react-native-svg | SVG для графіків |
| TypeScript | Типізація |

## Скріншоти

| Головна | Додавання | Статистика |
|---|---|---|

## Встановлення та запуск

### Вимоги

- Node.js 18 або новіший
- npm
- Додаток **Expo Go** на iOS-пристрої

### Кроки

1. Клонуйте репозиторій:
   ```bash
   git clone https://github.com/YOUR_USERNAME/FinanceApp.git
   cd FinanceApp
   ```

2. Встановіть залежності:
   ```bash
   npm install
   ```

3. Запустіть проект:
   ```bash
   npx expo start
   ```

4. Відскануйте QR-код у додатку **Expo Go** на вашому iPhone.
